Understanding This Dashboard

Direct Attach UEs: Devices that connect directly to the network for the first time.

X2AP Handover UEs: Devices that move between cells using X2 interface (direct eNB-to-eNB).

S1AP Handover UEs: Devices that move between cells using S1 interface (via core network).

Status Indicators: ✅ Success = Complete journey | ❌ Incomplete = Issues detected | Double-click any UE number to view detailed journey.
