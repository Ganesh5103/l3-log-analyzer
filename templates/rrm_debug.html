How to Fix Missing RRM Data
  1. Ensure RRM files are uploaded: Upload RRM_EVENT_X.dbg* files along with L3_EVENT_X.dbg* files
  2. Check time correlation: RRM logs must occur within 3 seconds of L3 admission request
  3. Verify UE index: RRM logs must contain "Received uecc_ue_index [X]" matching the L3 UE index
  4. Check RRM format: Logs should follow format: DD.MM.YYYY HH:MM:SS.microseconds filename.c line message
  5. Re-analyze: After uploading RRM files, clear results and re-analyze to trigger correlation
